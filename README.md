## 🚀 Features

- 🎪 基于 **vitepress** 工具开发
- 🔋 完全使用 **Markdown** 语言，上手无难度

## 📦 Install
```bash
pnpm install
```

## 🦄  serve
```bash
pnpm run docs:dev
```

## 🧱  build
```bash
pnpm run docs:build
```
