## 🚀 Features

- 🎪 基于 **vue3** 框架开发
- ⚡ 基于 **vite** 开发，获取更快速的启动速度
- 🦾 完全使用 **typescript** 提供完备的代码类型检查
- 🔋 **unocss** 为本项目提供了 **attributify** 的css书写
